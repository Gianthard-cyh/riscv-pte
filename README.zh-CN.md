# riscv-pte 🚀

一款轻量级 RISC-V Sv39 页表项（PTE）可视化工具。

[**在线体验**](https://gianthard-cyh.github.io/riscv-pte/)

## 功能亮点 ✨

- **全面展示 Sv39 PTE，逐位详解**
- **交互式切换位状态，实时观察 PTE 变化 🔄**
- **一键复制全部 PTE 信息 📋**
- **支持暗色模式，护眼更舒适 🌙**
